# Skill: Design System KITAMO

## Consulte
Leia `docs/tokens.md` para todos os valores de cor, tipografia e espaçamento.

## Princípios
- Clareza antes de beleza
- Mobile-first (90% do uso é mobile)
- Touch targets mínimo 44x44px
- Estados vazios sempre: ícone + texto educativo + CTA

## Tom Visual
- Teal como cor de confiança e ação
- Verde como positivo/conquista
- Vermelho como alerta (não como punição)
- Copy amigável, nunca corporativo
- Tom "galera" — referência: Nubank
