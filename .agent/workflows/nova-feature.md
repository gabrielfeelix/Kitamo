---
description: Implementa uma nova feature completa no KITAMO do zero.
---
1. Ler inventario em .agent/skills/nova-feature/docs/inventario.md
2. Confirmar com o usuário que a feature não existe ainda
3. Definir: o que muda no banco? Quais modelos? Quais rotas?
4. Criar migration + model + controller + form request
5. Criar componente Vue ou página Inertia com TypeScript
6. Verificar consistência com design system
7. Atualizar inventario.md com a nova feature adicionada
